Automates the process of typing in codes to a single keystroke from a list of 10,000 4-digit combinations ordered by frequency

**Commands** <br />
[enter] ~ Enters code <br />
[+] ~ Jump forward code <br />
[-] ~ Go back a code <br /> <br /> <br />
